# 📊 Excel Data Analytics Portfolio 🚀

## 📌 Overview
This repository showcases my complete learning journey in Microsoft Excel for Data Analytics and Business Intelligence.

It covers everything from basic Excel operations to advanced data analysis techniques, focusing on real-world applications such as data cleaning, transformation, analysis, and dashboard creation.

---

## 🎯 Objectives
- Understand Excel as a Business Intelligence Tool
- Learn how to structure and manage data effectively
- Perform data analysis using formulas and functions
- Build interactive dashboards and reports
- Gain practical skills required for a Data Analyst role

---

## 🛠️ Tools Used
- Microsoft Excel  
- GitHub (for version control & documentation)

---

## 📚 Topics Covered

### 🟢 Basics
- Excel Interface & Navigation  
- Workbook & Worksheet  
- Rows, Columns, and Data Structure  
- Data Types  

### 🟡 Data Handling & Formatting
- Data Entry Techniques  
- Formatting (Font, Colors, Borders)  
- Alignment, Wrap Text, Merge Cells  
- Sorting & Filtering  

### 🔵 Formulas & Functions
- Arithmetic Operations  
- Aggregate Functions (SUM, AVERAGE, COUNT, MAX, MIN)  
- Text Functions (LEFT, RIGHT, MID, TEXTBEFORE, TEXTAFTER)  
- Logical Functions (IF, AND, OR, XOR)  

### 🟣 Advanced Excel
- Lookup Functions (VLOOKUP, HLOOKUP, XLOOKUP)  
- PivotTables & PivotCharts  
- Slicers & Timeline Filters  

### 🔴 Data Visualization
- Charts (Bar, Line, Pie, Donut)  
- Dashboard Creation  

### ⚫ Data Cleaning & Other Features
- TRIM & Data Cleaning Techniques  
- Error Handling (IFERROR, ISERROR)  
- Data Validation  
- Sheet Protection & Sharing  
- Hyperlinks & Collaboration  
- Printing & Page Setup
